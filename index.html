Manga Downloader

Paste a WeebCentral link, pick chapters, get a ZIP or PDF. Runs entirely in your browser — nothing is uploaded anywhere.

This site talks to WeebCentral directly from your browser. That only works if WeebCentral allows it (a setting called CORS). If it doesn't, this app automatically routes the request through a free public proxy instead — but those are shared, rate-limited, and sometimes just offline. That's the most common reason a fetch fails.

Try this first

  1. Just hit Fetch again — public proxies often fail for a few seconds, then work.
  2. Wait 30–60 seconds if it keeps failing (you may be rate-limited).
  3. Open the Advanced log below the progress bar — it shows exactly which step and which proxy failed, which helps if you report a bug.

Reliable fix: use your own free proxy (2 minutes, no coding)

  1. Open Cloudflare Workers and sign up free:
    https://workers.cloudflare.com/
    Open it in a new tab →
  2. Create a new Worker and replace its code with this, then click Deploy: 
    export default {
  async fetch(request) {
    const target = new URL(request.url).searchParams.get('url');
    if (!target) return new Response('Missing url param', { status: 400 });
    const upstream = await fetch(target, {
      headers: { 'User-Agent': 'Mozilla/5.0' }
    });
    const body = await upstream.arrayBuffer();
    return new Response(body, {
      status: upstream.status,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Content-Type': upstream.headers.get('Content-Type') || 'text/plain'
      }
    });
  }
};
  3. Copy your Worker's URL (looks like https://something.yourname.workers.dev) and paste this template below, replacing the domain part:
    https://YOUR-WORKER.workers.dev/?url={url}

Leave this blank to just use the public fallback proxies.
